What is API

API(Application Programming Interface) is allow to communicate two applications each other. Api is simplified way to communicate the cloud -native app also some time time to share data to the end users.

In mordrtn world time is money if is easyest way to communicat using API from different device. Api can be private or publick in private api we need acc key to access. In private API increase the business value bucaue allow to share data to the user or other partners

Restful APIs use HTTP verbs

  • GET - requests using GET should retrive data from database
  • POST - POST method submit data to the database
  • PATCH - PATCH method apply partial modifications to the resources
  • DELETE - DELETE method delete record to the database
